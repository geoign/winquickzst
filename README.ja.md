# WinQuickZst

*言語: [English](README.md) | **日本語***

フォルダを右クリックし、好きな保存先へ高速な `tar.zst` を直接作成するWindows専用ツールです。同梱のRustエンジンがtar生成とZstandard圧縮をオンザフライで行うため、中間tarファイルや後からの移動はありません。

## 特徴

- フォルダ右クリック →「**WinQuickZst で tar.zst に圧縮…**」→ 圧縮率を選択 → 保存先を選択。
- 圧縮レベルにかかわらず、利用可能な全論理CPUをZstandardワーカーに使用します。
- 1 MiB以上のファイルは再利用可能な8 MiB入力バッファで高速に読み込み、小さいファイルには軽量な読み込み経路を使用します。
- 圧縮中は処理済み容量、出力容量、実効速度、経過時間を1秒ごとに同じ行へ表示します。
- 出力名は `フォルダ名_YYYYMMDD-HHMMSS.tar.zst` です。
- 指定フォルダ本体をアーカイブ最上位に含めます。
- Zstandardフレームチェックサムをオンザフライで付与します。
- UIはWindowsの表示言語・地域に応じて日本語または英語になります。`WQZ_LANG=ja` / `WQZ_LANG=en` でも指定できます。

## 選べる圧縮率

| UI表示 | Zstandardレベル | 用途 |
|---|---:|---|
| 高速（既定） | 2 | 転送時間を最優先 |
| 標準 | 10 | サイズと時間のバランス |
| 高圧縮 | 19 | 処理時間よりサイズを優先 |

## 動作要件

- 64ビット版Windows 10 / 11
- 圧縮対象はフォルダのみ
- 管理者権限・追加ランタイム・ネットワーク通信は不要

## インストール

1. このフォルダを任意の場所へ配置します。クラウド同期フォルダの場合は「このデバイス上に常に保持する」に設定してください。
2. **`Install.cmd`** をダブルクリックします。
3. Explorer再起動後、フォルダの右クリックメニューに「WinQuickZst で tar.zst に圧縮…」が表示されます。

## 使い方

- **フォルダ1つ**：フォルダを右クリック →「WinQuickZst で tar.zst に圧縮…」
  - Windows 11で見当たらない場合は「その他のオプションを表示」（`Shift + F10`）を確認してください。
- **複数フォルダ**：複数選択 → 右クリック → **送る** → **WinQuickZst**
- 圧縮率を選び、続いて保存先フォルダを選択します。

自動実行も可能です。

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\WinQuickZst.ps1 -Level std -Dest "D:\out" "C:\data"
```

`-Level` は `light`（Lv.2）、`std`（Lv.10）、`max`（Lv.19）です。

## このツールがPCに加えるもの

現在ユーザー（HKCU）だけに以下を登録します。管理者権限は使いません。

- フォルダ右クリックメニュー1件
- 「送る」ショートカット1件

圧縮は同梱の `bin\fast-tarzst.exe` だけで実行します。常駐処理、データ収集、ネットワーク通信はありません。

## Linuxでの展開

Linux側には `zstd` コマンドが必要です。

```bash
sudo apt-get install zstd
tar --zstd -xf folder_YYYYMMDD-HHMMSS.tar.zst
```

## アンインストール

1. **`Uninstall.cmd`** をダブルクリックします。
2. このフォルダを手動で削除します。

## 注意事項

- 進捗表示では圧縮元の事前走査を行いません。割合や残り時間ではなく、実際の処理状況を表示します。
- 処理中に圧縮元が変更された場合、アーカイブ全体の一貫性は保証されません。
- 停電やプロセス強制終了では、途中の `.tar.zst` が残る場合があります。
- NTFS ACL、代替データストリーム、Unix所有者・実行ビットの完全保存は対象外です。
- 高圧縮Lv.19は多くのCPU・メモリ・時間を使用します。
